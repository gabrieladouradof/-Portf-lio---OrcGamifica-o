Projetos

InovaAI

O InovaAI foi um projeto criado para empreendedores e fascinantes do mercado de inovação e startups. Desenvolvido pela disciplina de orientação a objetos no curso de Engenharia de Software da UnB, para inserir em prática conceitos de orientação a objetos e padrões de projeto. O projeto foi desenvolvido com Python, HTML, CSS e o Framework Flask, e utiliza Inteligencia Artificial para gerar conteudos valiosos para os usuarios.

StarTech

Este foi um projeto desenvolvido em curso de Desenvolvimento Web Fullstack, onde desenvolvemos um E-commerce Mobile, o StarTech. Todo o UX/UI Design foi desenvolvido por mim, utilizando o Figma. A aplicação foi desenvolvida utilizando a linguagem Dart e o Framework Flutter, além do Banco de Dados Oracle. O projeto foi desenvolvido em grupo, como um dos projetos finais do programa de formação TodasTech, da Imá Tech em parceria com a Novo Mundo.
